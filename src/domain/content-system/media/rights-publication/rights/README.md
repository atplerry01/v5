# rights (SCAFFOLD — pending implementation)

## Purpose

The `rights` leaf records **rights intent** on a media asset — territory, window, rightsholder attribution, usage permissions. Not authoritative legal grant (which lives upstream); this domain records the claimed rights state relevant to media distribution.

## Owns (planned)

- Rights record identity, asset ref, territory, window, rightsholder ref, permissions set, status.
- Claim / revise / expire / supersede transitions.

## Does not own

- Legal enforcement — upstream legal layer.
- Commercial entitlement — commerce layer.

## Status

SCAFFOLD only in P2.6.CS.10.
