# package (SCAFFOLD — pending implementation)

## Purpose

The `package` leaf owns **named packages** of media assets — album, episode pack, bundle, collection. Like `document/core-object/bundle` but for media.

## Owns (planned)

- Package identity, name, member set (opaque asset refs), status.
- Create / rename / add-member / remove-member / finalize / archive transitions.
