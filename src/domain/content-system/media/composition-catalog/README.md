# media / composition-catalog

## Purpose

Groups **composition and packaging** domains — grouping multiple assets into packages and sequencing them for ordered consumption.

## Leaf domains

- `package/` — (SCAFFOLD) named package of media assets (album, episode pack, bundle).
- `sequence/` — (SCAFFOLD) ordered sequence of assets for playlist/series consumption.

## Status

SCAFFOLD only in P2.6.CS.10.
