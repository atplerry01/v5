# Clusters Domain

## 📌 Purpose

Defines **system topology and economic structure**.

It is responsible for:
- Cluster definition
- Authority structure
- Subcluster organization

---

## 🧠 Core Invariants

- Every subcluster belongs to a cluster
- Authority must be defined
- Structure must be consistent

---

## 🚫 Forbidden

- Orphan subclusters
- Undefined authority

---

## 🔒 WBSM v3 Compliance

- Structural backbone of system
