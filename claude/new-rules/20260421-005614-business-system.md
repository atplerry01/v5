## RULES

### BS-INV-PRESENT-01 (S1)

Every aggregate under `src/domain/business-system/**` MUST declare
`private void EnsureInvariants()`.

**Scan:** pattern `private\s+void\s+EnsureInvariants\s*\(\s*\)` on
`*Aggregate.cs` under the business-system root. Missing match = violation.

### BS-INV-NON-TRIVIAL-01 (S2)

Every `EnsureInvariants()` body MUST contain at least two `throw` statements.
Identity-only validation is insufficient — every aggregate must additionally
assert required relationships, lifecycle state, or state-transition
preconditions.

**Scan:** count of `\bthrow\b` inside the method body < 2 = violation.

### BS-AGG-PURITY-01 (S1)

Aggregate files MUST NOT import other aggregate namespaces. Only the shared
kernel (`Whycespace.Domain.BusinessSystem.Shared.*`) is permitted as a
cross-folder business-system import. Cross-aggregate rules belong in
DomainPolicy classes under `policy/`, not inside aggregates.

**Scan:** in `*Aggregate.cs` under business-system, any
`using Whycespace.Domain.BusinessSystem.<X>;` where `<X>` does not start with
`Shared.` = violation.

### BS-EVENT-NAMING-01 (S2)

Every file in `*/event/*.cs` under business-system MUST:
- end with `Event.cs`,
- contain exactly one `public sealed record <FileNameWithoutExtension>(...)`
  declaration matching the filename.

### BS-EVENT-PAYLOAD-01 (S1)

Event record parameter lists MUST NOT contain `Guid *Id` or `string *Id`
primitive relationship fields. Typed canonical IDs (`AggregateId`, `*Ref`)
only.

### BS-POLICY-DECISION-01 (S1)

`*Policy` classes under `*/policy/*.cs` MUST NOT expose any `public void`
methods. Every public method returns a decision object — policies do not
mutate state.

### BS-POLICY-PURITY-01 (S0)

`*Policy` files MUST NOT reference any of: `Repository`, `DbContext`,
`IEventStore`, `IProducer`, `IConsumer`, `.SaveAsync`, `.SaveChanges`. Policies
receive aggregate state as value inputs and return pure decisions. No I/O, no
persistence, no messaging.

### BS-PRIM-ID-PROP-01 (S1)

No file under `src/domain/business-system/**` may declare a
`public (Guid|string) *Id` property. Every identifier is a typed VO.

### BS-PRIM-ID-PARAM-01 (S1)

No file under `src/domain/business-system/**` may declare a
`(Guid|string) *Id` method or constructor parameter. Primitive IDs are
forbidden on the wire as well as at rest.
