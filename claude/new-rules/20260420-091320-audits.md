## What the aggregate actually represents

State: PlaybackId + SourceRef + Mode + Window + Status + timestamps.
Behaviour: Create, Enable, Disable (with reason), UpdateWindow, Archive.
Events: Created, Enabled, Disabled, WindowUpdated, Archived.

The aggregate tracks, for a given source (stream / session / recording), in a
given mode (on-demand / catchup / replay), during a given window, whether
playback is ENABLED or DISABLED. It is a boolean availability gate with a
window, not a session, not a replay instance, not a manifest.

       PlaybackStatus, PlaybackWindow — carry over with PlaybackX → AvailabilityX
