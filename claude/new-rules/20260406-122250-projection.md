CLASSIFICATION: projection
SOURCE: System validation — TodoProjectionBridge type mismatch
DESCRIPTION: The EventFabric passes raw domain events (e.g. `TodoCreatedEvent` from `Whycespace.Domain`) as `EventEnvelope.Payload`, but projection bridges match on shared contract schema types (e.g. `TodoCreatedEventSchema` from `Whyce.Shared.Contracts`). These are different types — no case matches and the event is silently dropped.
PROPOSED_RULE: Projection bridges MUST match on the same types that the domain layer emits. Either: (a) EventFabric maps domain events to schema types before dispatching, or (b) projection bridges match on domain event types directly. Type alignment between emit and consume MUST be verified at build time or registration time.
SEVERITY: S1
