**PROPOSED RULE:** E1XD-AGG-LIFECYCLE-COMPARISON-01 (S1) — When a lifecycle-init guard compares `Version` against a literal integer, and `Version` is a custom value-object type, the comparison must use `Version.Value >= 0` or define `implicit operator int` on the VO. **REMEDIATION:** In `EventSchemaAggregate.cs:23` and `EventDefinitionAggregate.cs:25`, change `Version >= 0` to `Version.Value >= 0`. Alternatively, add `public static implicit operator int(EventSchemaVersion v) => v.Value;` to the `EventSchemaVersion` VO.
