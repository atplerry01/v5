

Recommended governing interpretation
Foundational principle

Each phase must produce a working vertical slice, not just code completion.

Delivery rule

For every domain after Phase 2, implementation should follow:

E1 → EX → Platform API proof

That means:

domain model
contracts/events
persistence/infrastructure
engine execution
runtime binding
system orchestration
platform exposure
projection/read model
policy/guard/chain enforcement where applicable
API validation and end-to-end confirmation
